# FormatFlow Free Tools Roadmap

## Truly free forever approach
Use browser-only processing wherever possible. No API keys. No server cost. No usage limits except the user's device/browser.

## Safe browser-only tools to build
- Image converter: PNG, JPG, WEBP
- Images to PDF
- Merge PDF
- Split PDF
- Resize image
- Compress image
- Crop image
- Rotate image
- Add watermark
- Remove metadata from images
- PDF page reorder
- PDF rotate pages
- Extract PDF pages

## AI tools note
Background removal, image enhancement, AI upscaling and object cleanup usually need hosted AI models or paid APIs. Avoid pretending these are free forever unless self-hosted later.

## Best next build
Add resize image, compress image, crop image, rotate image and add watermark. These are free, browser-based and good for traffic.
